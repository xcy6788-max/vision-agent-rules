# vision-agent-rules
这是一个专为 Claude Code CLI 设计的高级系统提示词（System Prompt）配置文件。它通过引入“主从代理隔离（Sub-agent Isolation）”与“严格熔断机制（Circuit Breaker）”，彻底解决了大模型在处理复杂工程时容易出现的“上下文污染”、“无脑读取全盘”以及“陷入终端报错死循环（Death Loop）”等痛点。

✨ 核心特性

物理级记忆隔离：强制主模型将繁重的代码检索（Grep/Glob）和报错排查（Bash）任务交由子代理在独立后台执行，保证主对话窗口极度纯净。

防卡死熔断器 (3-Strike Circuit Breaker)：严禁模型在遇到终端环境报错时无限重试。设定最高 3 次的试错红线，一旦越线立刻向人类开发者拉响警报，极大地节省了 API Token 并避免了系统假死。

去拟人化调度：摒弃自然语言伪指令，强制大模型通过标准的底层 Tool-Use（工具调用）去隐式唤醒子代理。

🛠️ 安装与使用方法

放置文件：将本仓库中的配置文件下载，并重命名为 CLAUDE.md。
